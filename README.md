# Personal Portfolio

A modern, responsive personal portfolio website with a Node.js backend.

## Features
- **Frontend**: Responsive HTML/CSS/JS with smooth animations.
- **Backend**: Express.js server for handling contact form submissions.
- **Database**: Supports MySQL and SQLite.
- **CI/CD**: Automated workflows via GitHub Actions.

## Getting Started
1. Clone the repository.
2. Install dependencies: `cd "portfolio backend" && npm install`.
3. Set up your `.env` file with database credentials.
4. Run the server: `npm start`.

## CI/CD Workflows
- **CI**: Runs linting and installation checks on every push.
- **CD**: Template for automated deployment to platforms like Render.
